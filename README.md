# 고분자대학원 실습 자료

Claude 세미나 및 실습용 저장소입니다. 루트에는 세미나용 **PowerPoint** 두 개가 있고, 하위 폴더는 **1차 실습**과 **2차 실습** 구역으로 나뉩니다.

## 폴더 구성

| 폴더 | 실습 구분 | 설명 |
|------|-----------|------|
| [`adhesives_data/`](./adhesives_data/) | **1차 실습** | 접착제·건재 관련 표준/번역 문서, 논문 PDF, 요약·카탈로그, 견적·매출 등 참고 데이터 |
| [`2nd/`](./2nd/) | **2차 실습** | 견적서 생성 스크립트(`1quotation`, `3quotation_re`), 플롯 예제(`2plot`), 논문 기반 문서화 파이프라인(`4paper`) 및 Claude Code 보조 설정(`.claude`) |

## 루트 파일

- `1고분자_Claude_세미나_기초활용_업무지원.pptx` — 세미나 1차(기초·업무 활용)
- `2고분자_Claude_Code_2회차_실전워크플로우.pptx` — 세미나 2차(실전 워크플로)

## 사용 시 참고

- 대용량 바이너리·문서가 포함되어 있으므로 클론 시 용량과 시간을 고려하세요.
- `2nd/` 일부는 Claude Code용 로컬 설정(`.claude`)을 포함합니다. 팀과 공유할 때는 민감 정보(API 키 등)가 없는지 확인하세요.
